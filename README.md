# beetslib

code that i use to manage my personal [beets](https://beets.io/) library

this is heavily specific for my usecase but if you want to use it for your own things its licensed under cc0

only config option is `opusdir` defaults to `/root/Music`

`uv tool install --with-executables-from beetcamp --with "beetslib@git+https://github.com/electron271/beetslib" "beets[chroma]"`
